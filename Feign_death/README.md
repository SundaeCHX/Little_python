# Feign_death

系统假死“病毒”，已打包为 Windows 可执行程序，为了伪装使用了 NVIDIA 的图标和命名。

双击 NVIDIA.exe 程序，程序会无界面直接启动，锁死键盘、鼠标（鼠标可移动不可点击），同时程序会自动设置为开机自启，每次开机都会造成系统假死。

按 “F12” 即可结束并退出程序。

源代码见 Feign_death/NVIDIA.py 。
